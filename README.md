# MEO - Avatar - Creator

## Get Started

### 1. 克隆项目

```bash
git clone https://github.com/givingwu/meo-avatar-creator.git
cd meo-avatar-creator
```

### 2. 安装依赖

使用你喜欢的包管理器（npm、yarn 或 pnpm）：

```bash
# 使用 npm
npm install

# 或者使用 yarn
yarn

# 或者使用 pnpm
pnpm install
```

### 3. 启动开发服务器

```bash
# 使用 npm
npm run dev

# 或者使用 yarn
yarn dev

# 或者使用 pnpm
pnpm dev
```

开发服务器启动后，通常会自动打开浏览器访问 http://localhost:5173（或终端提示的端口）。

### 4. 构建生产环境代码

```bash
# 使用 npm
npm run build

# 或者使用 yarn
yarn build

# 或者使用 pnpm
pnpm build
```

### 5. 预览生产环境构建

```bash
# 使用 npm
npm run preview

# 或者使用 yarn
yarn preview

# 或者使用 pnpm
pnpm preview
```
